Other Stuff

The nice thing about a framework like this is that you can add other pieces as needed, with no worries about any kind of conflicts.

Math

To add MathJax (math) support, put this piece of code at the top of this page: 
<script>
MathJax = {
  tex: {
    inlineMath: [['$', '$'], ['\\(', '\\)']]
  },
  svg: {
    fontCache: 'global'
  }
};
</script>
<script type="text/javascript" id="MathJax-script" async
  src="https://cdn.jsdelivr.net/npm/mathjax@3/es5/tex-svg.js">
</script>

Markdown

You might not want to write your own html. You might also wish to avoid having to convert manually from markdown to html using a tool like Pandoc. Add something like markdown-it to enable markdown conversion in the browser. There are many different ways to do that, so you'll need to figure it out on your own.
